Build a programmatic video composition.

1. Load the video-compositor agent.
2. Determine the right tool (Remotion vs. Motion Canvas).
3. Load `skills/production-motion/SKILL.md`.
4. Define the composition structure (dimensions, fps, duration).
5. Build the scene/sequence hierarchy.
6. Implement transitions between sequences.
7. Add audio sync if specified.
8. Verify the rendering pipeline works.

Report: composition structure, tool selection rationale, render settings.

Build a creative/generative animation from a brief.

1. Load the creative-coder agent.
2. Determine rendering target (Canvas 2D / WebGL / SVG).
3. Load `skills/creative-animation/SKILL.md`.
4. Select the right library (p5.js, PixiJS, vanilla Canvas).
5. Load relevant reference docs for the pattern (particle systems,
   flow fields, generative patterns, etc.).
6. Implement the animation.
7. Add a pause/play control for accessibility.
8. Implement prefers-reduced-motion (static frame or reduced speed).

Report: what was built, rendering target choice, and interaction model.
